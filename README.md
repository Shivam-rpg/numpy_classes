# NumPy Learning Practice 📊

This repository contains **four Jupyter Notebook files** demonstrating the fundamental concepts of **NumPy**, a powerful Python library for numerical computing.

The notebooks cover array creation, indexing, operations, statistics, and matrix manipulation.

# 📚 Topics Covered

## 1️⃣ NumPy Array Creation (Class1file)

This notebook introduces different ways to create NumPy arrays.

Concepts covered:

* Creating arrays from Python lists
* Zeros and Ones matrices
* Full matrices
* Range arrays
* Identity matrix
* Random number generation

## 2️⃣ Array Properties and Indexing (class2file)

This notebook focuses on **array structure and accessing elements**.

Concepts covered:

* Shape of arrays
* Number of dimensions
* Array size
* Data types
* Indexing
* Slicing
* Boolean indexing
* Reshaping arrays
* Flattening arrays

## 3️⃣ NumPy Operations and Statistics (class3file)

This notebook explains **mathematical operations and statistical functions**.

Concepts covered:

* Element-wise operations
* Scalar operations
* Arithmetic operations between arrays
* Statistical functions

## 4️⃣ Advanced Indexing and Matrix Operations (class4file)

This notebook demonstrates **advanced NumPy techniques**.

Concepts covered:

* Fancy indexing
* Conditional filtering
* Sorting arrays
* Concatenation
* Vertical and horizontal stacking
* Matrix multiplication
* Transpose of matrix
